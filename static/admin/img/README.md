# Christlumen Logo Files

Place your Christlumen logo files in this directory:

## Required Files:

1. **christlumen-logo.png** - Main logo for the admin panel (recommended size: 200x60px)
2. **christlumen-icon.png** - Favicon/icon for the browser tab (recommended size: 32x32px)

## Logo Specifications:

- Format: PNG with transparent background
- Main logo: Should be horizontal, suitable for header display
- Icon: Should be square, simplified version of the logo
- Colors: Should complement the blue and gold color scheme
